Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

  ## What it does

  The popup exposes two on-demand actions for the current Google Maps tab:

  - Download the scraped data as JSON
  - Send the scraped data to a configurable REST API

  Scraping only runs after a popup button click.

  ## Setup

  1. Install dependencies:

  ```bash
  npm install
  ```

  2. Create a `.env` file if you want API posting enabled:

  ```bash
  VITE_API_URL=https://example.com/api
  ```

  3. Build the extension:

  ```bash
  npm run build
  ```

  ## Load unpacked

  1. Open Chrome and go to `chrome://extensions`
  2. Turn on Developer mode
  3. Click Load unpacked
  4. Select the `dist` folder generated by the build

  ## Notes

  - The extension is limited to Google Maps-related pages.
  - The content script only scrapes when the popup asks for it.
